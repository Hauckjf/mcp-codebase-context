# Security Policy

## Reporting a vulnerability

If you find a security issue, please email **contato@fabiodiashauck.com.br**
with the subject line `SECURITY: <repo>` and a clear description of the
problem.

Do **not** open a public issue for security reports. We aim to acknowledge
within 72 hours and provide a fix or mitigation within 14 days.

## Supported versions

The `main` branch is supported. Tagged releases follow semantic versioning;
patch releases for the latest minor are supported.
